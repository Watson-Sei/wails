# Hello World

このチュートリアルの目的は、Wailsを使った最も基本的なアプリケーションを立ち上げ、実行することです。以下のことができるようになります:

- 新しいWailsアプリケーションを作成する
- アプリケーションをビルドする
- アプリケーションを実行する

:::note

このチュートリアルでは、ターゲットプラットフォームとしてWindowsを使用しています。出力はお使いのOSによって若干異なります。

:::

## Wailsの新規アプリケーションを作成する

デフォルトのVanilla JSテンプレートを使って新しいWailsアプリケーションを作成するには、次のコマンドを実行する必要があります。

```bash
wails init -n helloworld
```

以下のようなものが表示されるはずです:

```
Wails CLI v2.0.0

Initialising Project 'helloworld'
---------------------------------

Project Name:      helloworld
Project Directory: C:\Users\leaan\tutorial\helloworld
Project Template:  vanilla
Template Support:  https://wails.io

Initialised project 'helloworld' in 232ms.
```

これにより、カレントディレクトリには`helloworld`という新しいディレクトリが作成されます。このディレクトリの中に、いくつかのファイルがあります:

```
build/       - Contains the build files + compiled application
frontend/    - Contains the frontend files
app.go       - Contains the application code
main.go      - The main program with the application configuration
wails.json   - The project configuration file
go.mod       - The go module file
go.sum       - The go module checksum file
```

## アプリケーションをビルドする

アプリケーションをビルドするには、新しい`helloworld`プロジェクトディレクトリに移動して、以下のコマンドを実行します:

```bash
wails build
```

以下のような画面が表示されるはずです:

```
Wails CLI v2.0.0

App Type:               desktop
Platforms:              windows/amd64
Compiler:               C:\Users\leaan\go\go1.18.3\bin\go.exe
Build Mode:             Production
Skip Frontend:          false
Compress:               false
Package:                true
Clean Build Dir:        false
LDFlags:                ""
Tags:                   []
Race Detector:          false

Building target: windows/amd64
------------------------------
  - Installing frontend dependencies: Done.
  - Compiling frontend: Done.
  - Generating bundle assets: Done.
  - Compiling application: Done.
Built 'C:\Users\leaan\tutorial\helloworld\build\bin\helloworld.exe' in 10.616s.
```

これで、アプリケーションがコンパイルされ、`build/bin`ディレクトリに保存されました。

## アプリケーションを実行する

Windowsエクスプローラーで`build/bin`ディレクトリを表示すると、プロジェクトのバイナリが表示されるはずです:



```mdx-code-block
<div class="text--center">
  <img
    src={require("@site/static/img/helloworld-app-icon.webp").default}
    width="134px"
  />
</div>
<br />
```

`helloworld.exe`ファイルをダブルクリックするだけで実行できます。

Macでは、Wailsは`helloworld.app`ファイルを生成し、それをダブルクリックすることによって実行できます。

Linuxでは、`build/bin`ディレクトリにある、`./helloworld`を使って実行することができます。

